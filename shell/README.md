# Shell 脚本示例

本目录包含 Shell 基础语法与流程控制示例，按主题分目录组织。

## 目录结构

```
shell/
├── operator/   # 运算符示例
├── proc_ctrl/  # 流程控制示例
├── variable/   # 变量与参数示例
├── test_file/  # 文件测试示例
└── test.sh     # 基础测试脚本
```

## 学习路径建议

1. variable/ → operator/ → proc_ctrl/
2. test_file/ → test.sh

## 运行方式

```bash
bash variable/string_var.sh
```
