# Go 语言示例

本目录包含 Go 语言基础语法与并发相关示例，文件按主题命名。

## 示例文件

```
variable.go   # 变量与常量
const.go      # 常量
if.go         # 条件判断
for.go        # 循环
switch.go     # 分支
func.go       # 函数
defer.go      # defer
error.go      # 错误处理
pointer.go    # 指针
struct.go     # 结构体
method.go     # 方法
interface.go  # 接口
map.go        # 映射
slice.go      # 切片
array.go      # 数组
channel.go    # 通道
goroutine.go  # 协程
select.go     # select
recover.go    # panic/recover
package.go    # 包
hello.go      # Hello World
```

## 学习路径建议

1. hello.go → variable.go → const.go → if.go → for.go → switch.go
2. func.go → defer.go → error.go → pointer.go → struct.go → method.go → interface.go
3. array.go → slice.go → map.go
4. goroutine.go → channel.go → select.go → recover.go

## 运行方式

```bash
go run hello.go
```
