# 仓库总览与学习路线

## 结构
- /languages/<lang>：各语言的示例代码与测试，按章节组织
- /scripts：统一运行测试与工具脚本
- /docs：公共文档与章节索引
- /.github/workflows：CI 配置

## 章节索引（统一）
- ch01-basics：变量与类型、表达式与语句
- ch02-control：条件、循环与跳转
- ch03-functions：函数/方法与作用域
- ch04-ds：常见数据结构
- ch05-io：文件与标准 I/O
- ch06-error：错误/异常处理
- ch07-oop-generics：面向对象/泛型
- ch08-concurrency：并发/协程/线程
- ch09-modules：模块/包管理
- ch10-testing：测试约定与示例

## 使用建议
- 依次阅读各语言的 README 与章节示例
- 先跑通测试，再进行练习与扩展
