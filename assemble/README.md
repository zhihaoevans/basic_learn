# 汇编语言示例

本目录包含最基础的汇编语言示例，涵盖指令格式、循环控制与 C/ASM 混合编译，适合作为入门实践。

## 目录结构

```
assemble/
├── add.s           # 简单加法与寄存器示例
├── test.S          # 预处理汇编示例（.S 扩展名）
└── loop/
    ├── for.c       # C 语言循环对比
    └── for.asm     # 汇编循环实现
```

## 学习要点

- 了解 AT&T / Intel 语法差异（以当前示例为准）
- 熟悉寄存器、指令与寻址方式
- 对比 C 与汇编实现的控制流

## 运行方式（示例）

```bash
# 以 GNU 汇编为例
as add.s -o add.o
ld add.o -o add
./add
```

如需与 C 文件混编，可使用 gcc 直接编译链接。
